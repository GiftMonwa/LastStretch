Capstone Project Japanesse Writing Evaluator Project Description

The Japanesse Handwriting evaluator allows a user to upload an image of a symbol/word/sentence that they have written and detect which symbol it is supposed to be, the writing style (Kanji, Katakana or Hiragana), compare strokes and give the user points of improvement on the symbol(s) with a score of how well their symbol(s) matches the actual one
